# Save Recipe

Save recipes under `~/notes/recipes/` as a plain Markdown file with a short, slug-style filename.

Only save:

- the ingredient list
- the directions/instructions

# Recall Recipe

You may be asked to recall directions, instructions, or both, either as recorded, or modified. For examle, with the ingredients halved. For ingredient recall, use plain text, with a blank line between each ingredient.

# Edit Recipe

You may be asked to edit a recipe. For example, to "double the salt".
